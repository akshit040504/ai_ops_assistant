# AI Operations Assistant

## Overview
This project implements a multi-agent AI system that plans tasks, executes real API calls,
and verifies results using LLM reasoning.

## Agents
- Planner Agent: Converts natural language into a structured plan
- Executor Agent: Executes steps using real APIs
- Verifier Agent: Ensures correctness and completeness

## APIs Used
- GitHub API
- OpenWeatherMap API

## How to Run
1. Add API keys to .env
2. pip install -r requirements.txt
3. python main.py
